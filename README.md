## How to start and run it  
1. create a virtuel python enviroment and activate it `python -m venv venv && ./source/bin/activate`
2. create the database `python manage.py migrate`
3. create a superuser `python manage.py createsuperuser`
4. login as the superuser on 127.0.0.1/admin and create the categories and products - see video
